# MARK-6---SISTEMA-DE-NOTAS-DE-ALUNOS
Este projeto consiste no desenvolvimento de um Sistema de Notas de Alunos, criado em Java com foco na aplicação prática dos principais conceitos da (POO). O sistema simula o gerenciamento acadêmico básico de uma instituição de ensino, permitindo o cadastro de alunos, o registro de notas e o cálculo automático do desempenho escolar.
